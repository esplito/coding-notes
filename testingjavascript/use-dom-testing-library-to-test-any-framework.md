
# Use DOM Testing Library to test any JS Framework

Repo: [https://github.com/kentcdodds/dom-testing-library-with-anything/tree/main](https://github.com/kentcdodds/dom-testing-library-with-anything/tree/main)

-   React
-   Preact
-   jQuery
-   Dojo
-   HyperApp
-   AngularJS
-   Angular
-   VueJS
-   Mithril
-   from-html
-   Svelte
